# How do I run this?
## Setup
1. Create a virtual environment.
2. Source the virtual environment.
3. Run:
```bash
pip install -r requirements.txt
```

## Post-setup
1. Run:
```bash
python three-body.py
```
2. Enter your inputs via standard input.
